# Car Parts Point of Sale System

## Overview

The Car Parts Point of Sale System is a Python console application for an automotive parts store. The system allows cashiers to search for parts, filter parts by vehicle information, manage customer sales, process payments, generate receipts, and handle returns and refunds. 

## Features

The system supports the following use cases:

- Cashier logs in
- Search part by name or part number
- Filter parts by car year, make and model
- Start new sale
- Add part to cart
- Remove part from cart
- Update quantity
- Calculate tax and total
- Process cash payment
- Process card payment
- Process returns or refund
- Print receipt

## Technologies Used

- Python 3
- Visual Studio Code
- GitHub
